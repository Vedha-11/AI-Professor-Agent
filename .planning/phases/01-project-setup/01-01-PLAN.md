<title>Create FastAPI backend skeleton</title>

<action>
Create `backend/main.py` with minimal FastAPI application:

```python
"""
AI Professor Agent - FastAPI Backend
"""
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(
    title="AI Professor Agent",
    description="Course-agnostic AI tutoring system with RAG over course materials",
    version="0.1.0"
)

# CORS middleware for Streamlit frontend
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:8501"],  # Streamlit default port
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/")
async def root():
    return {"message": "AI Professor Agent API", "status": "running"}

@app.get("/health")
async def health_check():
    return {"status": "healthy"}
```

Verify server starts:
```
cd backend
uvicorn main:app --reload --port 8000
```

Test endpoint: `curl http://localhost:8000/health` should return `{"status": "healthy"}`
</action>

<action>
1. Create `frontend/app.py` with minimal Streamlit application:

```python
"""
AI Professor Agent - Streamlit Frontend
"""
import streamlit as st
import requests

st.set_page_config(
    page_title="AI Professor Agent",
    page_icon="🎓",
    layout="wide"
)

st.title("🎓 AI Professor Agent")
st.markdown("*Your course-agnostic AI tutor*")

# Backend health check
st.sidebar.header("System Status")

try:
    response = requests.get("http://localhost:8000/health", timeout=5)
    if response.status_code == 200:
        st.sidebar.success("✅ Backend: Connected")
    else:
        st.sidebar.error("❌ Backend: Error")
except requests.exceptions.RequestException:
    st.sidebar.warning("⚠️ Backend: Not running")

st.info("Welcome! This is the AI Professor Agent. Features will be added in upcoming phases.")
```

2. Verify Streamlit runs:
```
streamlit run frontend/app.py --server.port 8501
```

3. Verify Ollama is installed and working:
```

```

If Ollama not installed, document that user needs to install it from https://ollama.ai and pull a model:

