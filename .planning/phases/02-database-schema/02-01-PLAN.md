# Plan 02-01: Database Models and Setup

## Overview
Create SQLAlchemy models and database initialization for the AI Professor system.

## Tasks

### Task 1: Create database configuration
**File**: `backend/database.py`
- SQLite connection string
- SessionLocal factory
- Base declarative class
- Auto-create tables function

### Task 2: Create SQLAlchemy models
**File**: `backend/models.py`
- User model (id, username, password_hash, created_at)
- Course model (id, name, description, created_at)
- Material model (id, course_id, filename, filepath, uploaded_at)
- Submission model (id, user_id, course_id, content, score, submitted_at)

### Task 3: Initialize database on startup
**File**: `backend/main.py`
- Import and call create_tables on startup
- Add basic database health check

## Acceptance Criteria
- [ ] Database file created at `data/professor.db`
- [ ] All 4 tables exist with correct columns
- [ ] Foreign key relationships work
- [ ] Server starts without errors
